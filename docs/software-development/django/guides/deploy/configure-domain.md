# Configure Domain

## Buy on NameCheap

Click Manage

## Change nameservers

In Namecheap change nameservers to digitalocean ones like ns1.digitalocean.com ...

## In digitalocean project

Click on Create -> Domain

Do `@` and `www` point to the droplet IP
