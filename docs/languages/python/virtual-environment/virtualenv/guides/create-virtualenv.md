# Create VirtualEnv

## Init virtualenv

```
virtualenv my_env
```

If you have both python 2 and 3 installed:

```
virtualenv -p python3 my_env
```

## Activate

Linux:

```
source my_env/bin/activate
```

Windows:

```
my_env/Scripts/activate.bat
```
