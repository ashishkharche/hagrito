# Multiple Selections

## Alt + J

To select symbol.

## Alt + J again

Select next occurence of that symbol.

## Alt + Shift + J

To deselect the last occurence.

## Ctrl + Alt + Shift + J

To select all occurences in the file.

## Type

Type something and it will reflect on the selections.
